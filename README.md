In questo progetto ho creato un sito ecommerce denominato "MTB shop" con l'obiettivo di familiarizzare con il plug-in Woocommerce.
Come richiesto sono stati inseriti dei prodotti e categorie prodotto all'interno del sito web facilitando così la navigazione delle pagine.
Sono state inoltre create quattro zone di spedizione suddivise per CAP così da poter offrire un tariffario diverso per ogni luogo di destinazione 
della merce. Oltre alla semplice spedizione a tariffa unica, è stato creato l'opzione "ritiro in sede gratuito" che verrà visualizzato tra le opzioni di consegna 
disponibili solo se il cliente risiede nella stessa città del negozio.
Come richiesto è stato attivato il metodo di pagamneto "Bonifico".
